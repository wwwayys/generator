# Генератор отчетов об успеваемости студентов

### Данный проект представляет собой консольное приложение для формирования аналитических отчетов по успеваемости и посещаемости студентов на основе табличных данных.
## Цель:  
Программа создана в учебных целях, предназначена для автоматизации анализа учебных показателей, а также для практики работы с файлами,  структурирования кода и построения аналитических отчетов. Может использоваться в демонстрационных целях.
## Средства:
В качестве входных данных используется **CSV-файл**, содержащий информацию о студентах и их результатах за учебный период, а также библиотека **pandas** для **Python**. Приложение обрабатывает эти данные, вычисляет основные показатели и выводит сведения по запросу в удобном для восприятия виде.

## Программа поддерживает следующие функции:
- Расчёт средней общей оценки для каждого студента
- Расчёт среднего балла среди студентов по конкретным предметам
- Поиск студента с самой лучшей и самой худшей успеваемостью 
- Поиск студента с самой лучшей и самой худшей посещаемостью
- Интерактивное меню для навигации по функциям
- Поиск информации о студенте по ID

## Проект состоит из следующих модулей:
- **Main.py**
> *Точка входа в программу: запускает приложение и инициализирует основной цикл работы.*
- **Menu.py**
> *Отображение и обработка меню: отвечает за взаимодействие с пользователем и выбор действий.*
- **Controller.py**
> *Управление логикой приложения: связывает пользовательский интерфейс, аналитику и вывод результатов.*
- **DataLoader.py**
> *Загрузка и дополнение данных: загружает данные из CSV-файла, дополняет их и преобразует в структуру, удобную для анализа.*
- **Analytics.py**
> *Анализ и вычисление показателей: содержит функции для расчета средней оценки, посещаемости и рейтингов.*
- **Prints.py**
> *Форматированный вывод информации: реализует вывод отчетов и таблиц в консоль.*
- **Students.csv**
> *Исходный датасет.*

## Принцип работы
Программа функционирует в интерактивном режиме. Пользователь выбирает пункт меню, после чего данные автоматически обрабатываются и выводятся в виде отчета. По завершении операции пользователь может вернуться в главное меню, а затем завершить работу приложения.


Дополнительный функционал (индивидуальные правки)
Василиса - поиск студентов по ID
Эта функция отлично подходит для нашего приложения, является очень полезной для пользователя и поэтому часто фигурирует в подобных проектах. 
Благодаря этой функции пользовательский опыт становится разнообразнее, а функционал шире.
Эти правки прекрасно соотносятся с нынешней структурой проекта, гармонично дополняя модуль Analytics.py.
